## 22. 部署 PHP 网站

### 22.1 自定义网络

```
docker network create lnmp
```

### 22.2 创建 PHP 容器

```
docker run -itd \
--name lnpm_php
--name lnpm \
--mount type=bind,src=/app/wwwroot,dst=/usr/local/nginx/html \
php:v1
```

### 22.3 创建 nginx 容器

```
docker run -itd \
--name lnmp  \
--net lnpm \
-p 8888:80 \
--mount type=bind,src=/app/wwwroot,dst=/usr/local/nginx/html \
nginx:v1
```

### 22.4 创建 mysql 容器

```
docker run -itd \
--name lnmp_mysql
--net lnmp \
--mount type=bind,src=mysql-vol,dst=/usr/lib/mysql \
-e MYSQL_ROOT_PASSWORD=123456
mysql --character-set-server=utf8
```
